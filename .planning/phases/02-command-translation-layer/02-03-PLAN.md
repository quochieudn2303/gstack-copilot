## Tasks

### Task 1: Process Substitution Transform
**File:** `src/pipeline/transforms/process-substitution.ts`

Handle the `source <(cmd)` pattern that appears in every gstack preamble:

**Bash pattern:**
```bash
source <($GSTACK_BIN/gstack-repo-mode 2>/dev/null) || true
REPO_MODE=${REPO_MODE:-unknown}
```

**PowerShell equivalent:**
```powershell
$_repoModeOutput = & $env:GSTACK_COPILOT_BIN\gstack-repo-mode.ps1 2>$null
if ($?) {
    # Parse and set variables from output
    $_repoModeOutput -split "`n" | ForEach-Object {
        if ($_ -match '^(\w+)=(.*)$') {
            Set-Variable -Name $Matches[1] -Value $Matches[2] -Scope Script
        }
    }
}
if (-not $REPO_MODE) { $REPO_MODE = 'unknown' }
```

Implement:
1. Detect `source <(...)` patterns
2. Extract the command inside `<(...)`
3. Generate explicit two-step PowerShell:
   - Capture output to variable
   - Parse key=value pairs
   - Set as script-scoped variables
4. Handle `|| true` fallback pattern

### Task 2: Default Value Pattern
**File:** `src/pipeline/transforms/shell-idioms.ts`

Translate Bash default value syntax:

| Bash | PowerShell |
|------|------------|
| `${VAR:-default}` | `if (-not $VAR) { $VAR = 'default' }` |
| `${VAR:=default}` | `if (-not $VAR) { $VAR = 'default' }` |
| `${VAR:+alt}` | `if ($VAR) { $VAR = 'alt' }` |

### Task 3: Shell Conditional Patterns
**File:** `src/pipeline/transforms/shell-idioms.ts` (extend)

Translate common shell conditional patterns:

| Bash | PowerShell |
|------|------------|
| `[ -f "$file" ]` | `Test-Path $file -PathType Leaf` |
| `[ -d "$dir" ]` | `Test-Path $dir -PathType Container` |
| `[ -n "$var" ]` | `$var -and $var.Length -gt 0` |
| `[ -z "$var" ]` | `-not $var -or $var.Length -eq 0` |
| `[ "$a" = "$b" ]` | `$a -eq $b` |
| `[[ "$str" =~ regex ]]` | `$str -match 'regex'` |
| `cmd && next` | `if ($?) { next }` |
| `cmd || fallback` | `if (-not $?) { fallback }` |

### Task 4: Redirection Patterns
**File:** `src/pipeline/transforms/shell-idioms.ts` (extend)

| Bash | PowerShell |
|------|------------|
| `2>/dev/null` | `2>$null` |
| `>/dev/null 2>&1` | `*>$null` |
| `cmd > file` | `cmd | Out-File file` |
| `cmd >> file` | `cmd | Add-Content file` |
