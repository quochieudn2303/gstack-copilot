# Plan 02-02: UNIX Utility Translation

**Phase:** 2 - Command Translation Layer  
**Plan:** 2 of 3  
**Requirements:** TOOL-01, TOOL-02

## Objective

Create a JSON-based command mapping registry and implement translation for common UNIX utilities found in gstack skills to their PowerShell equivalents.

## Tasks

Create declarative JSON registry for command translations:

### Task 3: Command Transform Stage
**File:** `src/pipeline/transforms/commands.ts`

Implement transform function that:
1. Identifies Bash command invocations in code blocks
2. Matches against registry patterns (with placeholder extraction)
3. Generates equivalent PowerShell
4. Handles piped commands (`cmd1 | cmd2`)
5. Preserves commands not in registry with `# TODO: translate` comment

### Task 4: Complex Command Patterns
**File:** `src/mappings/commands.json` (extend)

Add patterns for compound commands found in inventory:
- `find ... -exec ... \;`
- `grep -r ... | wc -l`
- `ls -la`
- `[ -f {file} ]` → `Test-Path {file}`
- `[ -d {dir} ]` → `Test-Path {dir} -PathType Container`
